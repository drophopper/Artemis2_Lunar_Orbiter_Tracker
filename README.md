# Artemis II - Ultimate Mission Control Dashboard 🚀

Ein interaktiver Echtzeit-Tracker und Flugsimulator für die kommende **Artemis II Lunar Mission**. Dieses Dashboard visualisiert die Flugbahn, Geschwindigkeit und kritische Telemetriedaten der Orion-Kapsel auf ihrem Weg zum Mond.

## 🛰️ Features
* **Echtzeit-Telemetrie:** Anzeige von Geschwindigkeit (Mach), Höhe und Treibstoffstand.
* **Dynamische Visualisierung:** Animierte Orbit-Darstellung und Starfield-Effekt.
* **Responsive Design:** Optimiert für Desktop und mobile Endgeräte (ideal für das T450s).
* **Live-Status:** Überwachung der Triebwerkssysteme und Kabinenatmosphäre.

## 🛠️ Technik
Das Dashboard wurde mit modernem **HTML5, CSS3 und Vanilla JavaScript** entwickelt. Es nutzt mathematische Modelle zur Simulation der Mondtransfer-Trajektorie.
